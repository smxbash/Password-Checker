# Password-Checker
This script shows if the password is secure or not

It validates if the password contains:

- At least 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character


## How to Run
```chmod +x password-checker.sh
```
```
./password-checker.sh
```

## Requirements

- Linux operating system

- Bash shell

## What I Improved with this project

- Regex basics

- Input validation

- User interaction in terminal

## Project Goal

This project is part of my personal learning journey in Linux and Bash scripting.
It focuses on building logic and understanding how input validation works in real scenarios.

#### Notes

This script does not store or change system passwords.
It only checks if a password follows the defined rules.
